# EasyTGWhiteListMC

Telegram-бот для управления whitelist Minecraft сервера через RCON.

Подходит для:
- Локального запуска
- Запуска на хостинге
- Белый IP не требуется

Бот работает с любым сервером, где есть поддержка RCON.

---

# 🚀 Возможности

- Подача заявок на whitelist
- Автоматическая проверка шаблона заявки
- Фильтрация ника (A-Z, a-z, 0-9, _)
- Разделение лицензии и пиратки
- Привязка Telegram ID к нику
- Локальная база игроков
- Управление whitelist через RCON
- Админ-система
- Отправка сборки (.mrpack)
- Выполнение серверных команд через Telegram
- Система жалоб (v1.0.4+)

- Донат-система с промокодами (v1.0.5+)
- Смена ника, цвета ника и префикса через промокоды (v1.0.5+)
- Поддержка `.env` для конфигурации (v1.0.5+)

# 📝 Система заявок

Заявка должна содержать **ровно 7 строк**:

```
Заявка
Ник в Minecraft
Лицензия / пиратка
Возраст
Откуда узнали о сервере
Чем будете заниматься на сервере
Почему выбрали наш сервер
```

## 🔎 Проверки

- Ник: только `A-Z a-z 0-9 _`
- Строго 7 строк
- Если шаблон нарушен — заявка отклоняется
- Фильтрация запрещённых слов в нике (v1.0.5+)

---

## 📌 Пример корректной заявки

```
Заявка
Errnick_
Лицензия
16
Discord
Играть и помогать новичкам
Выбрал сервер за дружелюбную атмосферу
```

После отправки:
- Бот отвечает на сообщение игрока
- Появляется заявка
- Только администратор может принять или отклонить её

# 📝 Система жалоб (v1.0.4+)

Игроки могут отправлять жалобы на других игроков.

### Как написать жалобу

```
Жалоба
<ник игрока>
<причина>
```

**Пример:**

```
Жалоба
Errnick_
Гриф
```

### Как работает

1. Игрок отправляет жалобу — создаётся карточка.
2. Админ может **рассмотреть** жалобу.
3. После рассмотрения:
   - Автор жалобы видит, что она рассмотрена ⚡
   - Автор может:
     - **Закрыть** жалобу ✅
     - **Возобновить** жалобу ↩️ (ещё не решена)
4. Возобновлённая жалоба снова активна — админ может пересмотреть.

### Оформление карточки
```
📄 Жалоба
От: <кликабельный ник автора>
На: <кликабельный ник цели>
Причина: <текст жалобы>
❗ Статус: 🔴 Не решена / ⚡ Рассмотрена / ещё не закрыта автором
```

### Кнопки управления

- `Рассмотреть ⚡` — для админа
- `Закрыть ✅` / `Оставить открытой ↩️` — для автора жалобы

> Только игрок, который создал жалобу, может закрыть или возобновить её.
> Только админ может пометить жалобу как рассмотренную.

# 🎫 Система тикетов (v1.0.5+)

Игроки могут создавать тикеты в поддержку прямо в чате.

### Как создать тикет

```
тикет <описание проблемы>
```

**Пример:**
```
тикет Не могу зайти на сервер, пишет Not whitelisted
```

### Как работает

1. Игрок пишет `тикет <текст>` — создаётся карточка тикета.
2. Кулдаун между тикетами: **3 минуты**.
3. Только игрок с привязанным ником может создать тикет.
4. Адмнин нажимает **Рассмотреть ⚡**.
5. После рассмотрения автор может:
   - **Закрыть** тикет ✅
   - **Возобновить** тикет ↩️

### Оформление карточки
```
📄 Тикет
От: <кликабельный ник>
📝 <описание>
❗ Статус: 🔴 Новый / ⚡ Рассмотрен
```

# 🎁 Донат-система и промокоды (v1.0.5+)

Бот поддерживает выдачу и активацию промокодов для донатов.

### Типы промокодов

| Тип | Описание |
|-----|----------|
| `подписка` | Подписка "Плюс" (1 / 3 / 6 мес.) |
| `ник` / `ник5` / `ник10` | Смена отображаемого ника (1 / 5 / 10 раз) |
| `префикс` / `префикс5` / `префикс10` | Смена префикса (1 / 5 / 10 раз) |
| `цвет_<цвет>` | Смена цвета ника (например `цвет_красный`) |

### Цены

| Услуга | 1 раз | 5 раз | 10 раз |
|--------|-------|-------|--------|
| Смена ника | 350 руб. | 1500 руб. | 2800 руб. |
| Смена префикса | 250 руб. | 1100 руб. | 2000 руб. |
| Цвет ника | 150 руб./цвет | — | — |
| Подписка | 250 руб./мес. (-5% от 3 мес., -10% от 6 мес.) | — | — |

### Активация промокода

```
/promo КОД
```
или команда `промокод` / `промо` в чате.

### Генерация промокода (для админов)

```
генерировать ник
генерировать префикс5
генерировать цвет_красный
```
Или через интерактивную кнопочную панель `/admin`.

# 🎨 Смена ника, цвета и префикса (v1.0.5+)

### Смена отображаемого ника `/setnick`

```
/setnick &cPro Gamer
```

- Поддерживаются коды цветов Minecraft: `&0-9`, `&a-f`, HEX `&#RRGGBB`
- Длина ника без цветовых кодов: 2–16 символов
- Кулдаун: **30 минут** между сменами
- Требует активированный промокод типа `ник`

### Смена цвета ника `/nickcolor`

```
/nickcolor
```

Выбор из купленных цветов через кнопки. Требует промокод `цвет_<цвет>`.

### Смена префикса `/setprefix`

```
/setprefix [PRO]
```

- Поддерживаются цветовые коды
- Длина префикса без кодов: 1–8 символов
- Требует промокод `префикс`

---

# 🛡 Фильтр никнеймов (v1.0.5+)

При подаче заявки и смене ника автоматически проверяется:

- Формат: только `A-Z a-z 0-9 _`, длина 3–16 символов
- Чёрный список слов (оскорбления, имперсонация и пр.)
- Нечёткое сравнение через триграммы (схожесть > 75% — блок)
- Нормализация символов (0→o, 1→i, 3→e и т.д.)

Показывает игроков из локальной базы бота.

Особенности:
- Ник привязан к Telegram ID
- Ники кликабельные
- Есть ссылка на профиль Telegram

# 🔍 Команда "Проверить <ник>"

Проверяет:
- Есть ли игрок в базе
- Привязан ли к Telegram
- Есть ли в whitelist на сервере

Возможные статусы:
- Есть в базе и на сервере
- Только в базе
- Только на сервере
- Не найден
